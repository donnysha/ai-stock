## 约束规则遵守

```
┌─────────────────────────────────────────────────────────────────┐
│                        约束规则检查清单                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. AI层职责限定                                                   │
│     ✅ IntentClassifier - 只做意图识别                            │
│     ✅ ParamExtractor - 只做参数抽取                             │
│     ❌ 不做计算、不写SQL、不做回测                                 │
│                                                                 │
│  2. 业务逻辑层固化                                                │
│     ✅ SQLGenerator - 固化SQL生成规则                             │
│     ✅ StockSelector - 固化选股逻辑                              │
│     ✅ GridBacktester - 固化回测逻辑                              │
│     ❌ 不允许AI生成SQL或策略                                       │
│                                                                 │
│  3. 输出协议标准化                                                │
│     ✅ 统一使用 OutputSchema                                      │
│     ✅ 固定字段: intent, status, params, missing_fields           │
│     ❌ 不输出多余解释                                             │
│                                                                 │
│  4. 参数约束                                                     │
│     ✅ 缺失字段统一放入 missing_fields                            │
│     ❌ 不允许AI自行编造默认值                                      │
│                                                                 │
│  5. 模型配置                                                     │
│     ✅ temperature=0                                            │
│     ✅ 纯JSON输出                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
