## 核心功能

### 1. 自然语言选股

输入示例：
```
"帮我找出成交额大于5000万的科技行业股票，价格在10-50元之间"
"市盈率小于20的银行股"
"近5日涨幅大于10%的股票"
```

### 2. 网格交易回测

输入示例：
```
"对000001进行回测，下跌3%买入，上涨5%卖出，从2024年1月开始"
```

### 3. 涨跌停判断

支持各板块涨跌停规则：
- 主板（60/00/001开头）：±10%
- 科创板（688开头）：±20%
- 创业板（300开头）：±20%
- 北交所（8开头）：±30%

## 技术栈

| 分层 | 技术/工具 | 用途 |
|------|----------|------|
| 开发语言 | Python 3.x | 核心开发语言 |

## 使用方式

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动应用

```bash
# 方式1: 使用main.py
python main.py

# 方式2: 直接使用streamlit
streamlit run ai_stock_nlp/presentation_layer/app.py

# 方式3: 指定端口
streamlit run ai_stock_nlp/presentation_layer/app.py --server.port=8501
```

### 配置说明

在 `config/settings.py` 中配置：

```python
# AI模型配置
AI_CONFIG = {
    'model': 'gpt-4o-mini',
    'api_key': 'your-api-key',
    'temperature': 0,  # 固定为0，确保输出稳定
}

# 回测配置
BACKTEST_CONFIG = {
    'initial_capital': 1000000,  # 默认初始资金100万
    'commission_rate': 0.0003,  # 佣金费率万三
}
```

## 约束规则

1. **AI层职责限定**
   - IntentClassifier: 只做意图识别
   - ParamExtractor: 只做参数抽取
   - ❌ 不做计算、不写SQL、不做回测

2. **业务逻辑层固化**
   - SQLGenerator: 固化SQL生成规则
   - StockSelector: 固化选股逻辑
   - GridBacktester: 固化回测逻辑
   - ❌ 不允许AI动态生成SQL或策略

3. **输出协议标准化**
   - 统一使用OutputSchema
   - 固定字段: intent, status, params, missing_fields
   - ❌ 不输出多余解释

4. **参数约束**
   - 缺失字段统一放入missing_fields
   - ❌ 不允许AI自行编造默认值

## 项目定位

纯工具类辅助软件，不提供荐股服务，不保证收益，仅为用户提供选股策略验证与数据参考。

## 风险提示

1. **数据风险**: AkShare数据源接口不稳定
2. **AI解析风险**: 自然语言理解可能存在偏差
3. **回测偏差风险**: 历史回测不代表未来收益
4. **合规风险**: 股票数据使用存在合规限制
